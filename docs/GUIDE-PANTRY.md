## Editing Items

### Quick Edit
1. Tap any item to open edit mode
2. Change name, quantity, category, or expiry date
3. Tap **Save** to confirm

### Editable Fields
- **Name** - Rename the item
- **Quantity** - Update amount
- **Unit** - pcs, kg, g, L, ml, etc.
- **Category** - Reassign category
- **Expiry Date** - Pick new date
